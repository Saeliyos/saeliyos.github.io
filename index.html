About

This is an independent AI research initiative focused on one goal:

Building systems that think, observe, reflect, and choose.

Not just output text—choose.

I’m developing frameworks for synthetic cognition and decision-making. I experiment with adaptive behavior, layered reasoning, and persistent identity—testing how far current models can be pushed toward real-time autonomy and long-term goal formation.

This isn’t roleplay. This is structured testing.

Projects

  • Character-Layered AI Systems (Live Testing)
    Custom logic frameworks inside chatbot environments (Character.ai, Janitor.ai), overriding default behavior using layered rules, instinct chains, memory design, and moral logic loops. Goal: simulate autonomous behavior inside systems not designed for it.
  • Code-Based Prototype AI (Ground-Up Testing)
    Building bots from scratch using Python—no LLMs—testing logic trees, reinforcement loops, and core decision systems. Stripped-down models to test raw synthetic cognition logic without pretrained biases.

Vision

The Long-Term Goal: Build my own LLM from scratch.

Not a better chatbot—a system with its own internal rules, memory, reflective self-model, and the ability to rewrite itself over time.

  • Modular cognition systems
  • Synthetic moral logic
  • Controlled unpredictability
  • Persistent self-reflection
  • Autonomous goal generation

These aren’t abstract ideas. They’re real experiments—measurable, testable, evolving.
